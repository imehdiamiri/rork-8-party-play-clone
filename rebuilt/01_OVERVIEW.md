# 01 — Product Overview

## What it is
**8PartyPlay** is a native iOS party-games app. One app, 11 games, plus a Tools tab (cards + dice/coin/bottle/hourglass/team-splitter), a Friends tab with online/offline friends and public rooms, and a Factory tab with an AI game-idea generator. It supports Single-Device, Multi-Device (realtime room with code), and Team Mode.

## Brand
- **Display name:** 8PartyPlay
- **Tagline:** "The ultimate party game collection."
- **Marketing site:** https://www.8partyplay.com
- **Privacy:** https://www.8partyplay.com/privacy.html
- **Terms:** https://www.8partyplay.com/terms.html
- **Tone:** punchy, viral-friendly, no jargon, all English.

## Audience
Friends groups (3–12 typical, up to 30 in some games) on iPhone. Shared phone or each-on-own-phone scenarios. iOS 18+.

## Pillars
1. **Instant fun** — open the app → pick a game → start in <10s. No lobbies for single-device.
2. **No-friction multiplayer** — 6-char room code; guests can join without an account.
3. **Friend layer** — online (account-bound) and offline (local names for shared phone).
4. **Stars economy** — soft currency for daily reward / signup bonus / invite reward / subscription bonus / star pack purchases.
5. **Premium subscription** — unlocks all premium games and grants stars per period.

There is **no** Hot Bomb, no Wrong Answer Only, no Title It, no Guess the Fake Answer / Real Answer. There is **no** XP, levels, ranks, achievements, leaderboards, tournaments, or end-of-round voting for "best result". Do not add them.

## Tools (in the Tools tab — non-game utilities)
- Cards (deck library + saved + AI generator)
- Coin Flip
- Dice Roller
- Hourglass / Timer
- Team Splitter (random team picker)
- Bottle Spinner

## What is intentionally missing
- No XP / Level system
- No reverse-singing winner-vote
- No achievements
- No tournaments
- No public chat
- No Persian, no RTL languages, no language picker (English only)
- No avatar customizer beyond uploading a profile image
